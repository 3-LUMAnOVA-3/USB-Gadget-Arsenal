# USB-Gadget-Arsenal
USB-Gadget
Une collection de configurations avancées pour transformer des cartes ARM (Raspberry Pi Zero, Luckfox) en périphériques USB composites dynamiques. Ce projet explore l'utilisation de FunctionFS et des modules standards Linux pour créer des disques durs intelligents chiffrés, des outils de pentesting éthique (BadUSB multifonction) et des émulateurs de webcams.
